# Bedolaga Blacklist Bot

Telegram-бот для быстрой проверки, есть ли пользователь в [blacklist проекта ban-vpn](https://github.com/Blin4ickUSE/ban-vpn).

Бот работает в **inline-режиме**: результат можно отправить в любой чат, группу или канал, не открывая диалог с ботом.

## Что умеет бот

- **Проверить себя** — показывает ваш Telegram ID, имя, username и статус в blacklist.
- **Проверить по ID** — проверяет, есть ли указанный числовой ID в списке.

Список обновляется при каждой проверке: бот загружает актуальный `blacklist.txt` из репозитория ban-vpn.

## Как пользоваться

1. Найдите бота в Telegram и откройте его профиль.
2. В любом чате введите `@username_бота` (подставьте имя вашего бота).
3. Выберите нужный вариант:

| Что ввести после `@бот` | Результат |
|-------------------------|-----------|
| Ничего (пустой запрос) | Проверка **вашего** аккаунта |
| Числовой ID, например `777000` | Проверка **этого** ID |

4. Нажмите на подсказку в списке — бот отправит готовое сообщение с результатом.

### Примеры

- `@bedolaga_blacklist_bot` — проверить себя.
- `@bedolaga_blacklist_bot 777000` — проверить пользователя с ID `777000`.

В сообщении с проверкой себя есть кнопка **«Проверить себя»** — она обновляет статус без нового inline-запроса.

### Статусы

- **Не найден в blacklist** — ID нет в списке.
- **В BLACKLIST** — ID есть в списке.
- **Ошибка загрузки** — не удалось получить список (проблема с сетью или источником). Попробуйте позже.

## Откуда берётся blacklist

Бот использует публичный список из репозитория [Blin4ickUSE/ban-vpn](https://github.com/Blin4ickUSE/ban-vpn). В каждом ответе есть ссылка на этот репозиторий.

## Запуск своей копии (для администраторов)

Если вы разворачиваете бота сами:

1. Создайте бота через [@BotFather](https://t.me/BotFather) и включите **Inline Mode**.
2. Склонируйте репозиторий и установите зависимости:

```bash
npm install
```

3. Создайте файл `.env` по образцу `.env.example`:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
```

4. Запустите:

```bash
npm run dev
```
